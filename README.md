# Rest API server

`ypack` Rest API server.

## Requisites
To start hacking you need:
* [Golang 1.10](https://golang.org/)
* [Dep](https://github.com/golang/dep)

## Install
Clone this repo and build it:

```bash
$ go get -v github.com/ypack/ypack-server
$ cd cd ypack-server
$ go build
```
And now execute it.
